Build a CLI application to manage a set of markdown notes in powershell.

. Notes are stored in a folder identified by an env variables
. You need to implement CRUD operation as parameter to the main program (i.e., notes add, notes, ...)
. You need to be able to search for notes and visualize notes either as lists or a single screenshot of all notes that contain text
. You can edit notes with an editor in the standard env var for editors
